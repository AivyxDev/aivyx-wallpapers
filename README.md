# Aivyx Wallpapers

🕯️ Official desktop wallpapers for the Aivyx ecosystem — aligned with the **Candlelit Kitchen** brand identity (v3).

## Wallpapers

| # | Name | Description |
|---|------|-------------|
| 01 | **Candlelit Kitchen** | A warm scene with candle, terracotta tiles, sage herbs, and plum bottle |
| 02 | **Spice Rack Abstract** | Flowing ribbons of all five brand colors swirling around a central flame |

## Brand Colors Used

All wallpapers use the official v3 palette:

- **Midnight Hearth** `#1a1a28` — Background
- **Candle Amber** `#f4a261` — Primary flame
- **Amber Glow** `#ffc864` — Inner flame core
- **Cream** `#f5ebdc` — Candle bodies, text
- **Terracotta** `#C4553E` — Warm red accent
- **Sage** `#4D8B6A` — Organic green
- **Wick Purple** `#8B6FBF` — AI-thinking inner core
- **Plum** `#7A65A6` — Secondary purple
- **Teal** `#3C8B80` — Cool accent
- **Sienna** `#8B5E3C` — Warm brown
